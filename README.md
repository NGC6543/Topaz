# Topaz (В РАЗРАБОТКЕ)

Система заметок. По плану пользователь сможет увидеть граф соединяющий похожие заметки.
В данный момент разрабатывается поиск.


# Запуск проекта:

Клонировать репозиторий и перейти в него в терминале:
```bash
git clone https://github.com/NGC6543/Topaz.git
```

```bash
cd Topaz
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить главный файл:
```bash
py topaz.py
```


# Стек технологий

- Python 3.12.8,
- PyQt6 6.10.0
